# Asthma Risk Prediction Using AI

This project predicts asthma hospital admissions using air pollution and weather data in Lahore, Pakistan.

## Models Used
- Linear Regression
- Random Forest
- XGBoost
- Support Vector Regression

## Data
Pollution data: PM2.5, PM10, NO2, SO2, O3, CO  
Weather data: temperature, humidity, wind speed  

## Goal
To build an early warning system for asthma risk during high pollution days.
